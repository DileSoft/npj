Channels -- RSS+ Aggregator
--------
Version 1.0
http://npj.ru/npjdev/modules/channelsmodule
--------

Модуль Channels -- это унифицированный агрегатор потоков сообщений для НПЖ.
Он предназначен для автоматической синдикации в пространство НПЖ информации 
из других систем посредством создания "трансляционных аккаунтов", в которые 
направляются "каналы".

Для полноценной работы Channels необходимо устанавливать его на узле, работающим 
под управлением NPJ R1.7 или выше.
  * http://npj.ru/npjdev/release -- текущий релиз НПЖ.

Если вы обнаружите какие-либо проблемы в работе модуля, пожалуйста,
опишите их в баг-трекере проекта НПЖ: http://npj.ru/bugs

--------
Installation:
--------

1. РАЗМЕЩЕНИЕ ДИСТРИБУТИВА

Содержимое дистрибутива необходимо скопировать в ту же папку, где установлено
ПО НПЖ. В случае возникновения конфликтов в папке "lib/" они должны разрешаться
в сторону более свежего дистрибутива дублированной библиотеки.

Папка "channels/" должна встать в один ряд с папками "core/", "lib/", "npj/".


2. ПОДКЛЮЧЕНИЕ МОДУЛЯ

Для подключения модуля необходимо разкомментировать следующую строку 
в файле config_modules.php:
  // include( "channels/npj_config.php" );


3. ИНИЦИАЛИЗАЦИЯ МОДУЛЯ

Для инициализации модуля необходимо выполнить остальные шаги, описанные
здесь: http://npj.ru/npjdev/modules/channelsmodule#installation


4. НАСТРОЙКА МОДУЛЯ (опционально)
Вы можете настроить поведение модуля, прочитав документацию
