Connecting htmlArea to NPJ
--------
http://npj.ru/npjdev/modules/htmlareamodule
--------

Данный модуль позволяет заменить на вашем узле WYSIWYG-редактор RTE на htmlArea v.3.0.

Дистрибутив htmlArea взят с сайта http://htmlarea.com и используется по их лицензии.

"Обвязка" дистрибутива в модуль НПЖ является лицензионно-свободной, потому как
смысла без этих двух компонентов не имеет.

Для полноценной работы модуля необходимо устанавливать его на узле, работающим 
под управлением NPJ R1.7 или выше.
  * http://npj.ru/npjdev/release -- текущий релиз НПЖ.

--------
Installation:
--------

1. РАЗМЕЩЕНИЕ ДИСТРИБУТИВА

Содержимое дистрибутива необходимо скопировать в ту же папку, где установлено
ПО НПЖ. В случае возникновения конфликтов в папке "lib/" они должны разрешаться
в сторону более свежего дистрибутива дублированной библиотеки.

Папка "htmlarea/" должна встать в один ряд с папками "core/", "lib/", "npj/".


2. ПОДКЛЮЧЕНИЕ МОДУЛЯ

Для подключения модуля необходимо разкомментировать следующую строку 
в файле config_modules.php:
  // include( "htmlarea/npj_config.php" );
