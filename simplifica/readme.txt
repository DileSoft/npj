Упрощаем НПЖ
--------
http://npj.ru/npjdev/modules/simplifica_module
--------

Данный модуль "скрывает" от пользователей (стараясь не блокировать) некоторую 
чрезмерно сложную функциональность НПЖ.

Рекомендуется использовать этот модуль с одноимённой "шкурой".

Для полноценной работы модуля необходимо устанавливать его на узле, работающим 
под управлением NPJ R1.9 или выше.
  * http://npj.ru/npjdev/release -- текущий релиз НПЖ.

--------
Installation:
--------

1. РАЗМЕЩЕНИЕ ДИСТРИБУТИВА

Содержимое дистрибутива необходимо скопировать в ту же папку, где установлено
ПО НПЖ. 

Папка "simplifica/" должна встать в один ряд с папками "core/", "lib/", "npj/".


2. ПОДКЛЮЧЕНИЕ МОДУЛЯ

Для подключения модуля необходимо разкомментировать следующую строку 
в файле config_modules.php:
  // include( "simplifica/npj_config.php" );


3. УСТАНОВКА ШКУРЫ "simplifica" ПО-УМОЛЧАНИЮ

В базе в db_prefix."profiles" меняется default value у поля "skin" => "simplifica"

Вы также можете принудительно изменить это значение у всех существующих пользователей.
Кроме того, стоит изменить у них же поле "skin_override" на то же значение.
